## Configurações iniciais

[🔼](#índice)

-   Fazer as instalações

    ```bash
    npm i
    ```

-   Fazer a conexão com o banco de dados através da extensão `mySQL`.

-   Rodar as querys do arquivo `poo.sql`.

-   Testar os endpoins no `Postman`, por exemplo.
